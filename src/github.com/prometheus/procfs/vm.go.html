The VM interface is described at https://www.kernel.org/doc/Documentation/sysctl/vm.txt Each setting is exposed as a single file. Each file contains one line with a single numerical value, except lowmem_reserve_ratio which holds an array and numa_zonelist_order (deprecated) which is a string
