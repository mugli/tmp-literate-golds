A EnvProvider retrieves credentials from the environment variables of the running process. Environment credentials never expire. Environment variables used: * Access Key ID: AWS_ACCESS_KEY_ID or AWS_ACCESS_KEY * Secret Access Key: AWS_SECRET_ACCESS_KEY or AWS_SECRET_KEY
type EnvProvider struct {
	retrieved bool
}
