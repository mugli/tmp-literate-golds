RefSpec is a mapping from local branches to remote references. The format of the refspec is an optional +, followed by <src>:<dst>, where <src> is the pattern for references on the remote side and <dst> is where those references will be written locally. The + tells Git to update the reference even if it isnâ€™t a fast-forward. eg.: "+refs/heads:refs/remotes/origin/*" https://git-scm.com/book/en/v2/Git-Internals-The-Refspec
