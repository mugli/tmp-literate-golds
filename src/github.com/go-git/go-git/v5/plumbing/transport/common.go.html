Package transport includes the implementation for different transport protocols. `Client` can be used to fetch and send packfiles to a git server. The `client` package provides higher level functions to instantiate the appropriate `Client` based on the repository URL. go-git supports HTTP and SSH (see `Protocols`), but you can also install your own protocols (see the `client` package). Each protocol has its own implementation of `Client`, but you should generally not use them directly, use `client.NewClient` instead.

Endpoint represents a Git URL in any supported protocol.
Protocol is the protocol of the endpoint (e.g. git, https, file).
User is the user.
Password is the password.
Host is the host.
Port is the port to connect, if 0 the default port for the given protocol wil be used.
Path is the repository path.
	Path string
}
