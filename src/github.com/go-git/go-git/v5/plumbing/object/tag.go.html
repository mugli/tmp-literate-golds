Tag represents an annotated tag object. It points to a single git object of any type, but tags typically are applied to commit or blob objects. It provides a reference that associates the target with a tag name. It also contains meta-information about the tag, including the tagger, tag date and message. Note that this is not used for lightweight tags. https://git-scm.com/book/en/v2/Git-Internals-Git-References#Tags
