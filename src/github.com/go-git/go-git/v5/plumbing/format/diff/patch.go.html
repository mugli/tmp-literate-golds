Patch represents a collection of steps to transform several files.
FilePatches returns a slice of patches per file.
Message returns an optional message that can be at the top of the Patch representation.
	Message() string
}
FilePatch represents the necessary steps to transform one file to another.
IsBinary returns true if this patch is representing a binary file.
Files returns the from and to Files, with all the necessary metadata to about them. If the patch creates a new file, "from" will be nil. If the patch deletes a file, "to" will be nil.
Chunks returns a slice of ordered changes to transform "from" File to "to" File. If the file is a binary one, Chunks will be empty.
	Chunks() []Chunk
}
