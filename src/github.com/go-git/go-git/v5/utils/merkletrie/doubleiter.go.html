A doubleIter is a convenience type to keep track of the current noders in two merkletries that are going to be iterated in parallel. It has methods for: - iterating over the merkletries, both at the same time or individually: nextFrom, nextTo, nextBoth, stepBoth - checking if there are noders left in one or both of them with the remaining method and its associated returned type. - comparing the current noders of both merkletries in several ways, with the compare method and its associated returned type.
type doubleIter struct {
	from struct {
		iter    *Iter
		current noder.Path // nil if no more nodes
	}
	to struct {
		iter    *Iter
		current noder.Path // nil if no more nodes
	}
	hashEqual noder.Equal
}

Remaining returns if there are no more noders in the tree, if both have noders or if one of them doesn't.
func ( *doubleIter) () remaining {
	if .from.current == nil && .to.current == nil {
		return noMoreNoders
	}

	if .from.current == nil && .to.current != nil {
		return onlyToRemains
	}

	if .from.current != nil && .to.current == nil {
		return onlyFromRemains
	}

	return bothHaveNodes
}
