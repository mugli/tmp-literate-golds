The node represents a file or a directory in a billy.Filesystem. It implements the interface noder.Noder of merkletrie package. This implementation implements a "standard" hash method being able to be compared with any other noder.Noder implementation inside of go-git.

Hash the hash of a filesystem is the result of concatenating the computed plumbing.Hash of the file as a Blob and its plumbing.FileMode; that way the difftree algorithm will detect changes in the contents of files and also in their mode. The hash of a directory is always a 24-bytes slice of zero values
