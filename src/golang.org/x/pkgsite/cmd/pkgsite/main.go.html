This is a work in progress. Pkgsite extracts and generates documentation for Go programs. It runs as a web server and presents the documentation as a web page. Usage: pkgsite [flag] # Load module from current directory. pkgsite [flag] [path1,path2] # Load modules from paths to memory. The flags are: -gopath_mode=false Assume that local modules' paths are relative to GOPATH/src -http=:8080 HTTP service address to listen for incoming requests on
