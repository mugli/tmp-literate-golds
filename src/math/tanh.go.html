The go code is a simplified version of the original C. tanh.c Hyperbolic tangent SYNOPSIS: double x, y, tanh(); y = tanh( x ); DESCRIPTION: Returns hyperbolic tangent of argument in the range MINLOG to MAXLOG. MAXLOG = 8.8029691931113054295988e+01 = log(2**127) MINLOG = -8.872283911167299960540e+01 = log(2**-128) A rational function is used for |x| < 0.625. The form x + x**3 P(x)/Q(x) of Cody & Waite is employed. Otherwise, tanh(x) = sinh(x)/cosh(x) = 1 - 2/(exp(2x) + 1).
